Para instalar en tu PC local, abre una terminal en la carpeta de tu proyecto y ejecuta:

```bash
npm install exceljs
npm install -D @types/exceljs
```

O si usas **bun**:
```bash
bun add exceljs
bun add -d @types/exceljs
```

O si usas **yarn**:
```bash
yarn add exceljs
yarn add -D @types/exceljs
```

**Resumen:**
- `exceljs` → Librería principal
- `@types/exceljs` → Tipos para TypeScript (como dependencia de desarrollo con `-D`)
